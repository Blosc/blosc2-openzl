https://github.com/Blosc/blosc2_openzl#readme

## What is it?

Blosc2-openzl is a dynamic codec plugin for Blosc2 that allows to compress
and decompress data using the OpenZL compression library.  For details, check out the
[project README](https://github.com/facebook/openzl).
